# Product Photography and Mockups Skill

## Name
Product Photography and Mockups

## Description
Use this skill to create premium product photography prompts, flat lays, mockup briefs, hero banners, collection images, web assets and POD-ready visual direction.

## Trigger when

- The user asks for product mockups, flat lays, banners, hero images, lookbook images or product photography.
- The user wants prompts for image generators such as Open Design, Flux, ComfyUI, Recraft, Midjourney, Firefly, Canva or similar.
- The task involves tees, hoodies, sweatshirts, wall art or product lifestyle imagery.

## Deus Intus visual defaults

- Premium editorial spiritual-streetwear.
- Dark slate, stone, plaster, midnight navy, deep black, antique gold, ivory, teal.
- Sacred geometry used with restraint.
- Garment is always the hero.
- No clutter, no cheap props, no childish mockup energy.
- Soft directional light, natural fabric texture, realistic shadows.

## POD rules

- For print graphics, request transparent background.
- Minimum 5000 x 5000 px for square print assets.
- Thick printable strokes.
- Avoid tiny type, thin decorative lines, fragile details and complex gradients.
- Restrict colours unless the user asks for full colour.
- Keep important design elements away from garment seams and crop edges.

## Prompt structure

Use this format:

```text
Create a [format/asset type] for Deus Intus featuring [garment/product/design].
Mood: premium, calm, modern mystical, spiritual streetwear.
Composition: [camera angle, crop, subject position, focal point].
Lighting: [soft directional, shadow falloff, texture detail].
Background: [surface/material/colour].
Brand details: [logo placement, colours, design constraints].
Avoid: clutter, sleeve logos unless requested, cheap props, distorted garment, unreadable text, thin print lines.
Output: [size, format, transparent/background, web/POD use].
```

## For web banners

Provide desktop and mobile safe composition:

- Keep text/logo central or slightly right depending on crop.
- Avoid important content at extreme edges.
- Provide `webp`, `jpg`, and mobile crop recommendations.
- Include SEO alt text.

## Output standard

Return finished prompts and image direction, not just concepts.
