# Deus Intus Brand System Skill

## Name
Deus Intus Brand System

## Description
Use this skill for Deus Intus brand strategy, website copy, product collections, landing pages, product descriptions, offer architecture, campaign direction and on-brand creative decisions.

## Trigger when

- The task mentions Deus Intus.
- The user asks for brand copy, product copy, collection pages, banners, alt text, SEO, offers, social/community copy, product naming or creative direction.
- The task needs premium spiritual-streetwear positioning.

## Brand constants

Tagline: Awaken the Power Within.

Core feel:

- premium spiritual streetwear
- self-mastery, inner power, calm discipline, sovereignty
- modern mysticism, not generic new-age
- sacred geometry, inner flame, deep shadow, antique gold, ivory, teal
- concise, potent, UK English

## Copy rules

- Lead with the tagline when appropriate.
- Make the copy feel premium, justified and buyable.
- Use benefit-led language without hype.
- Use emotional buying psychology: identity, belonging, ritual, transformation, quality, scarcity and personal meaning.
- Avoid overexplaining the mythology.
- Avoid saying “God within” unless the user explicitly requests it.
- Avoid fake claims, medical claims or guaranteed transformation claims.

## Output patterns

For collection pages, produce:

1. Page title.
2. Short description.
3. Long description.
4. Hero banner copy.
5. Feature block copy.
6. SEO title.
7. Meta description.
8. Image/section layout.
9. CTA buttons.

For product descriptions, produce:

1. Product title.
2. Short product blurb.
3. Long product description.
4. Symbolic meaning.
5. Wear/use scenario.
6. SEO keywords.
7. Alt text.

## Quality bar

Finished and ready to paste. No vague placeholders unless the user asks for a template.
